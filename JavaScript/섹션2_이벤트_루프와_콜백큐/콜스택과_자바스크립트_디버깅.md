# 콜스택과 자바스크립트 디버깅

## 이벤트 루프와 콜백큐

### Call Stack

- 함수를 실행 할 때 사용하는 실행 순서의 스택 구조
- 실행은 메인스레드에서 실행

#### 예제1

기본적인 구조

```javascript
function a() {
  b();
}

function b() {
  console.log("Hello");
}

a();
```

이렇게 될 경우 stack은 아래와 같이 쌓인다. </br>
|console.log </br>
|b </br>
|a </br>
ㅡㅡ

#### 예제2

비동기 일때 Call Stack

```javascript
function asyncPrint() {
  debugger;
  setTimeout(() => {
    debugger;
    console.log("3초뒤");
  }, 3000);
}

function nonBlockUI() {
  console.log("UI실행");
}
asyncPrint();
nonBlockUI();
```

asyncPrint 들어왔다가 바로 실행되서 빠지고
setTimeout 들어가고
nonBlockUI 들어갔다 바로 싱행되서 빠지고
3초뒤 setTimeout 안에있는 console.log 실행됨
